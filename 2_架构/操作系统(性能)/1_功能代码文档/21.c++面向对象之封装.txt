C++：面向对象
  1、三大特点：封装，继承，多态
 
  2、封装：外界访问类中信息（成员变量、成员方法）时，所提供访问权限
		公有封装：public   		外界自身都可访问
		私有封装：Private		外界不能访问自身都可访问
		保护封装：protected		外界不能访问自身都可访问
		
类：某一类实体的抽象，模板。  （结构体）
  
	类：属性（特征）+方法（功能）
		
	定义类：（定义一个类型）
		class 类名
		{
		
			属性:
		};


对象（实例）：类的实例化		（结构体变量）


定义对象：
	类名  对象名;
  
实例化：是由构造器来完成。 （分配空间并初始化）

构造器：是一个特殊的方法（函数）
	
	类名(void)
	{
		
	}
	PS：1、因为构造器只完成构造成员空间和初始化。
		2、当不定义构造器，编译器会添加一个无参构造器（默认构造器）
		3对象调用构造器：由实参类型决定 

作用域：能被引用的范围
	1、局部变量：函数内定义的变量
			作用域在函数内
	2、全局变量：函数外定义的变量
			作用域在文件内
	3、类作用域：在类内定义的成员变量
			在类内中
		
 
作用域运算符   ::
  	::全局变量
	
	类::成员变量  : 作用声明在成员变量的作用域在该类中
	
this指针：每一个对象都有一个this成员（隐藏），指向 自身空间
		this  指向自身空间  
			
			地址->成员
			变量名。成员
			
		类名* this = &self

初始化列表：只能应用于构造器  
		类名（）:成员变量(值),成员变量2(值)
		{
		
		}


 
类外定义：//表明函数 在 该类中的作用域中
类名::方法名()
{

}

	指针->成员
	内容.成员
  
  
对象数组：存放的元素都是对象
